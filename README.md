# StockEasy｜らくらく備品管理システム

**StockEasy** は、介護施設や病院などの現場で使用される備品（車いす・エアマット・歩行器など）を、簡単かつ効率的に管理できるWebアプリケーションです。ローカル環境（ブラウザ）で動作し、データは自動でブラウザに保存（localStorage使用）されます。

![StockEasy Dashboard](https://via.placeholder.com/800x400/4f46e5/ffffff?text=StockEasy+Dashboard)

## 🌟 特徴

- **💻 インストール不要**: ブラウザがあればすぐに使用開始
- **🔒 データセキュリティ**: データはローカルに保存、外部送信なし
- **📱 レスポンシブ対応**: スマートフォンでも快適に操作
- **👥 役割別アクセス**: 職員用・管理者用の2段階アクセス制御

## ✅ 主な機能

### 📋 備品管理
- **備品登録・編集・削除**: 写真付きで備品情報を追加・編集
- **詳細情報管理**: 型番、購入日、メンテナンス履歴等の記録

### 🔄 貸出・返却システム
- **リアルタイム状況把握**: 誰がどこで使用中かを瞬時に確認
- **簡単操作**: ワンクリックで貸出・返却処理
- **履歴管理**: 過去の貸出履歴を自動記録

### 📍 位置・状態管理
- **保管場所追跡**: 各備品の現在地を正確に把握
- **ステータス管理**: 使用中・待機中・メンテナンス中等の状態表示
- **アラート機能**: 返却予定日の通知

### 📊 データ管理
- **バックアップ機能**: JSON形式でのデータエクスポート
- **データ復元**: 他の端末やブラウザへのデータ移行
- **定期バックアップ**: 自動バックアップ機能（推奨設定）

### 🧑‍⚕️ 権限管理
- **職員モード**: 貸出・返却操作、備品検索
- **管理者モード**: 全機能（登録・削除・設定変更）

## 🚀 使用方法

### アクセス方法
1. **[https://fractiqlabs.github.io/stockeasy-v2/](https://fractiqlabs.github.io/stockeasy-v2/)** をブラウザで開く
   - 推奨ブラウザ: Google Chrome, Firefox, Safari（最新版）
2. ログイン方式を選択：
   - **職員としてログイン**: 貸出・返却操作
   - **管理者としてログイン**: 全機能利用（パスワード：`admin123`）

### 基本操作
1. **備品登録**（管理者のみ）
   - 「備品追加」ボタンをクリック
   - 必要情報と写真を登録
   
2. **貸出処理**
   - 備品を選択
   - 「貸出」ボタンで貸出先を指定
   
3. **返却処理**
   - 貸出中の備品を選択
   - 「返却」ボタンで返却処理完了

## 💾 データ保存について

### 保存方式
- **ローカルストレージ使用**: データはブラウザのlocalStorageに保存
- **自動保存**: 操作後即座にデータを保存
- **永続化**: 再読み込み後も状態を保持

### バックアップ・復元
```javascript
// データエクスポート
「データエクスポート」ボタン → .json ファイルをダウンロード

// データインポート  
「データインポート」ボタン → .json ファイルを選択してアップロード
```

### データ移行手順
1. 現在の環境で「データエクスポート」を実行
2. 新しい環境でStockEasyを開く
3. 「データインポート」で保存したファイルを読み込み

## 💡 開発の背景

特別養護老人ホームなどの医療・介護現場では、以下の課題がありました：

- **備品の所在不明**: 車いすや歩行器の場所がわからない
- **貸出状況の把握困難**: 誰が使用中か不明確
- **手書き管理の限界**: 紙ベースでは情報共有が困難
- **時間コスト**: 備品探しに多くの時間を消費

これらの課題を解決するため、職員・管理者の双方が直感的に操作できるシステムとして開発されました。

## 🆘 トラブルシューティング

### よくある問題

**Q: データが消えてしまった**
```
A: ブラウザのキャッシュクリアが原因の可能性があります。
   定期的なデータエクスポートを推奨します。
```

**Q: 画像が表示されない**
```
A: ブラウザの容量制限に達している可能性があります。
   不要な備品データを削除してください。
```

**Q: スマホで操作しにくい**
```
A: ブラウザを最新版に更新し、画面を横向きにして
   お試しください。
```
